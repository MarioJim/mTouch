Marvel prototype

Low level prototype assets

At the home screen the user is able to check the gestures assigned to the existing devices, look at his/her profile, add a new gesture or access the side-bar menu.

The user can log-in if he/she wishes in order to store the global settings of the application and access them from different devices.

A side-bar menu displays, here the user is able to add a new knocki device, look at his/her account info, edit existing devices, edit gestures or provide feedback.

To add a new device the user can enter whether its name or location with the id number and finally press sync to conclude the process.

When a new device is created no gestures are available, the user can generate them by pressing the "+" button on the bottom right corner.

The user creates the gesture by introducing on the screen the pattern that he/she wants to set to perform a specific action.

After the gesture is selected, the user can decide which action will be performed after tapping the knocki device.

The new device is ready and the gesture stays registered, if the user needs to add a new one ha can press the "+" button and repeat the process.
