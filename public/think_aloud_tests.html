Findings

  • Users usually get stuck in the main screen since it seems a loop and there is no screen indicator.
  • The last option that they interact with is the add device option
  • There is no way to edit or delete already stablished gestures
  • When creating a new gesture there is no option to correct it in case of wrong input, a button to accept the gesture is also missing
  • There is no status indication when the users create a new device, so they don't notice it
  • Some tasks aren't obvious to use or find
